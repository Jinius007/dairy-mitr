# PashuMitra — Stage 1

Stage 1 is the first complete milestone: chat, voice notes, live call, and read-aloud in Indian languages via Bhashini TTS.

## Included in Stage 1

- Text chat with streaming AI replies
- Voice notes (record → transcribe → reply → read aloud)
- Live call mode with on-screen transcript
- Bhashini TTS for read-aloud (chat + call)
- Mobile-friendly layout (no scroll to reach input)
- Auto-open first chat on launch
- Deployed on Vercel with Supabase backend

## Stage 1 completion fix

- **Full read-aloud**: Long replies are split into sentence chunks and played sequentially so the entire answer is spoken, not just the first 2–3 lines.
